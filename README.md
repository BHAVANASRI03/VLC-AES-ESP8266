# 🔐 VLC with AES-128 using ESP8266

## 📌 Overview
This project demonstrates secure data transmission using Visible Light Communication (VLC) with AES-128 encryption.

## ⚙️ Components
- ESP8266
- LED (Transmitter)
- Photodiode (Receiver)

## 🔄 Working
1. Data is encrypted using AES-128
2. LED transmits encrypted data via light
3. Receiver captures and decrypts data

## 🔐 Security
AES-128 ensures secure communication by encrypting data before transmission.

## 🚀 Applications
- Secure IoT communication
- Smart lighting systems
